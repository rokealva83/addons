Advanced search

You can select search domain: pages, blog posts, blog posts comments, customers info, job opportunities and products.

Also you can define if search will be case sensitive or not.

It also searches over translations depending on website current language.

Search results

Presented with images, link to content, dates and result text description with highlighted search hit. Supports Pagination.

Everything customizable through templates and inheritance. Fully translatable.
